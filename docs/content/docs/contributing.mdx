# Contributing Data

There are three ways to contribute market data.

## 1. Submit a pull request

If you are comfortable editing JSON, this is the fastest path.

1. Open the correct file in `data/states/<state-slug>.json`
2. Find the matching LGA
3. Add a new market object to that LGA's `markets` array
4. Open a pull request

```json
{
  "name": "Balogun Market, Lagos Island",
  "slug": "balogun-market",
  "coordinates": { "lat": 6.4541, "lng": 3.3947 },
  "added_by": "your-github-username"
}
```

## 2. Use the web form

Visit `/contribute` on the main app to submit a new market suggestion. The API converts that submission into a GitHub issue for review.

## 3. Open a GitHub issue

Use the repository's `Add a Market` issue template if you prefer not to edit the data files directly.

## Validation rules

- Slugs must be lowercase and hyphen-separated
- Market slugs must be unique
- Coordinates are optional, but must be valid numbers if supplied
- Market names should include the LGA name for disambiguation
