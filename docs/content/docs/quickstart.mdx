# Quickstart

The API is public and does not require authentication. All endpoints return JSON.

## Base URL

```bash
https://iya-oloja.pages.dev/api
```

## List states

```bash
curl https://iya-oloja.pages.dev/api/states
```

## Get one state

```bash
curl https://iya-oloja.pages.dev/api/states/lagos
```

## List markets with pagination

```bash
curl "https://iya-oloja.pages.dev/api/markets?limit=10&offset=0&order=asc"
```

## Search for markets

```bash
curl "https://iya-oloja.pages.dev/api/search?q=balogun"
```

## Submit a market suggestion

```bash
curl -X POST https://iya-oloja.pages.dev/api/contribute \
  -H "Content-Type: application/json" \
  -d '{
    "market_name": "Balogun Market, Lagos Island",
    "state": "Lagos",
    "lga": "Lagos Island",
    "lat": 6.4541,
    "lng": 3.3947,
    "description": "One of the largest markets in Lagos",
    "contributor_name": "your-github-username"
  }'
```
