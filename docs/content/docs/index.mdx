# Iya Oloja API

Iya Oloja is a free JSON API for discovering markets across Nigeria. It is designed for developers who want simple read access to market, LGA, and state data without authentication.

## What you can do

- List all states and the FCT
- Fetch a state together with its LGAs
- Fetch an LGA together with its parent state and markets
- Paginate through all markets
- Search by market, LGA, or state name
- Submit new market suggestions for review
