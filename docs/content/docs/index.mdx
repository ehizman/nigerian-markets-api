# Overview

Iya Oloja is a free JSON API for discovering markets across Nigeria.

## Base URL

```bash
https://iya-oloja.pages.dev/api
```

## Use the interactive reference

The main API experience lives in the generated `API Reference` section. Open any endpoint there to inspect parameters, example payloads, and run requests interactively.

## Endpoints

- `GET /api`
- `GET /api/states`
- `GET /api/states/{state}`
- `GET /api/lgas/{lga}`
- `GET /api/markets`
- `GET /api/search`
- `POST /api/contribute`

## Contribution flow

Use `POST /api/contribute` to submit a market suggestion for review, or open the main app's `/contribute` page if you want a form-based flow.
